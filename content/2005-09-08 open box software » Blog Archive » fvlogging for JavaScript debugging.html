fvlogging for JavaScript debugging

  • September 8th, 2005
  • Posted by d@vid

Using techniques that will be familiar to anyone who’s looked under the hood of TiddlyWiki’s personal content editing engine, David Miller has created fvlogging a JavaScript logging tool that reports in the page you’re building. I’m not sure I’ll get a chance to test it out on my current project which, apart from simple DIV-swopping and validation, uses more CodeBehind than JS, but it looks neat.

You can read all about JavaScript Logging at A List Apart, naturally. (Along with a redesign, ALA4.0 features the coolest t-shirts )
