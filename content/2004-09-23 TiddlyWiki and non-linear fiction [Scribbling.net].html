TiddlyWiki and non-linear fiction
An experiment in wiki-izing fictional narrative.

So my new web application crush is Jeremy Ruston's TiddlyWiki, a completely self-contained, single file client-side wiki application. TiddlyWiki lets you edit, show and hide linked blocks of text dynamically on one page, using Javascript and DHTML.

It's the coolest thing my browser's seen in a long time.

Anyway, my first impulse was to Tiddlerize a piece of fiction, to see what would happen. So I wrote a short descriptive essay, and created a TiddlyWiki along with the usual vanilla text version. I think the differences bring to light some interesting things about how we read and write the web.

See for yourself:

  • Sitter - the usual, linear text narrative
  • BabyDogSitter - the TiddlyWiki version
